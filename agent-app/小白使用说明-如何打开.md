# 宝尊风控 AI Agent — 小白如何打开（本机 Windows）

把下面这份说明当作「开箱步骤」即可：按顺序做，就能在浏览器里打开产品页面。

---

## 一、你需要先准备什么

| 项目 | 说明 |
|------|------|
| **电脑系统** | Windows 10 / 11（本说明针对 Windows 双击启动）。 |
| **Python** | 已安装 **Python 3.10 或更高版本**。安装时务必勾选 **「Add Python to PATH」**。 |
| **浏览器** | Chrome、Edge 等任意常用浏览器。 |
| **网络** | 能正常上网（第一次要下载依赖包；使用 AI 对话时要能访问 DeepSeek）。 |
| **DeepSeek 密钥** | 在 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并创建 **API Key**（形如 `sk-…`）。 |

> 没有 Python？先到 [python.org](https://www.python.org/downloads/) 下载安装，再回来继续。

---

## 二、怎么打开（最简单：双击脚本）

1. 用资源管理器打开文件夹：  
   **`宝尊agent方案`** → **`agent-app`**
2. 找到文件：**`一键安装并启动.bat`**
3. **双击**运行它。

### 第一次运行时会发生什么

- 脚本会自动：创建虚拟环境 → 安装依赖（可能几分钟）→ 若还没有配置文件，会从 **`.env.example`** 复制出 **`.env`** 并用**记事本**打开。
- 在记事本里找到一行：  
  `DEEPSEEK_API_KEY=`  
  在等号后面**粘贴你的密钥**，保存并关闭记事本。
- **再次双击** `一键安装并启动.bat`。

### 启动成功的标志

- 出现一个**黑色命令行窗口**，里面最后类似：服务运行在 **8800** 端口。
- **不要关掉这个黑窗口**，关掉就等于关服务器。

---

## 三、在浏览器里「打开」产品

1. 打开浏览器。
2. 在地址栏输入下面任一地址并回车：

   - **http://127.0.0.1:8800/**
   - **http://localhost:8800/**

3. 若首次进入，页面会要求填写 **DeepSeek API Key**（可与 `.env` 里的一致），保存后即可进入主界面。

---

## 四、常用操作提醒

| 操作 | 做法 |
|------|------|
| **下次还要用** | 同样双击 `一键安装并启动.bat`，等黑窗口起来后再打开浏览器访问上面地址。 |
| **一键演示** | 进入产品后，顶部有 **「一键演示」** 按钮（需先完成 API 配置进入主界面）。 |
| **改密钥** | 可点界面里的 **「设置」**，或编辑 `agent-app` 目录下的 **`.env`** 后重启脚本。 |

---

## 五、常见问题（一眼能懂）

| 现象 | 可能原因 | 你可以怎么做 |
|------|----------|----------------|
| 提示找不到 python | 没装 Python 或没加 PATH | 重装 Python 并勾选 Add to PATH，重开窗口再双击脚本。 |
| pip 安装失败 | 网络慢或被墙 | 换网络、开代理，或配置国内 pip 镜像后再试。 |
| 浏览器打不开页面 | 服务没起来或端口被占 | 确认黑窗口还在、无报错；或换浏览器再试 `127.0.0.1:8800`。 |
| 对话报错 / 无回复 | 密钥错误或未填 | 检查 `.env` 与页面设置里的 Key 是否正确、是否有余额。 |

## 六、和「阿里云部署」的区别

- **本文件**：在自己电脑上 **双击 bat → 浏览器打开**，适合试用、演示、开发。
- **服务器上给全公司用**：请看同目录下的 **`阿里云部署说明.md`**（需要阿里云账号与 Docker 等）。

---

**文件位置（方便你转发给别人）**  
与本说明同级目录应包含：`一键安装并启动.bat`、`requirements.txt`、`server.py` 等。  
把整个 **`agent-app`** 文件夹拷给对方，让对方先看本 **`小白使用说明-如何打开.md`** 再双击 bat 即可。

## 七、从钉钉下载的压缩包怎么用

若你拿到的是 **`宝尊风控Agent-钉钉分享.zip`**（从钉钉文档里下载）：

1. **解压**到本机任意文件夹（例如桌面或 D 盘）。
2. 打开解压后的 **`宝尊风控Agent`** 文件夹。
3. 先看 **`【先看这里】解压后双击一键安装并启动.txt`** 或 **`小白使用说明-如何打开.md`**。
4. 双击 **`一键安装并启动.bat`**，再用浏览器打开 **http://127.0.0.1:8800/**。

压缩包**不含**虚拟环境 `.venv`（体积大），第一次运行脚本会自动创建；**不含** `.env` 密钥，需按提示自行填写。

**自己重新打一个压缩包上传到钉钉**：在 `agent-app` 里双击 **`打包钉钉分享.bat`**（需已装 Python），会在**桌面**生成 **`宝尊风控Agent-钉钉分享.zip`**。
